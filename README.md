# AI Agent Skill - Engineering Tutor

As AI agents continue to improve in writing code, novice engineers risk giving up on learning the fundamentals.
Expert engineers risk losing their proficiency if they give up thinking through solving architectural and design problems.

This skill is designed to help both novice and expert engineers stay sharp by providing a structured way to practice and improve their engineering skills.
